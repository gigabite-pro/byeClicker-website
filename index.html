ByeClicker

Automatically answers iClicker poll questions for you!

Say Bye to iClickers!

ByeClicker is a web application that automatically answers iClicker poll questions for you. It uses a machine learning model to predict the correct answer to the question and submits it for you. This saves you time and allows you to focus on other things during class.
